## 项目介绍

桔子云（JuziCloud），常被称为桔子云加速服务，是一家专注于 **SSR / V2Ray 协议** 的优质机场平台，隶属于速鹰体系的成熟老站。目前平台提供近 **50 个高质量付费节点**，线路统一采用 **CN2 / BGP 隧道中转**，并辅以 **IPLC 国际专线**，在速度、稳定性和延迟控制方面表现出色。

无论是需要大量流量的影音娱乐用户，还是仅在特定时期使用的轻度用户，亦或对流量消耗要求不高的用户，都能在 JuziCloud 找到匹配自身需求的套餐方案，整体定价合理，性价比突出。

桔子云早期曾提供过免费公益节点服务，后因带宽与维护成本不断上升，现已停止公益运营，全面升级为 **纯付费、高品质线路机场**，专注长期稳定运行。
